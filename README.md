# Web前端学习笔记

基于 VitePress 构建的个人前端技术学习笔记。

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建静态站点
npm run build

# 预览构建结果
npm run serve
```
